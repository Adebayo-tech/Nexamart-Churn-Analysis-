## Files

**Cleaned_Dataset.xlsx**  
Main dataset used for analysis. Cleaned customer data ready for processing.

**Dashboard(Nexamart_Churn_Analysis).xlsx**  
Excel dashboard showing churn patterns. Helps track key metrics visually.

**Nexamart_Complaint_Analysis.xlsx**  
Detailed complaint analysis. Identifies main reasons customers are unhappy.

**Nexamart_presentation-1.pptx**  
Project findings summary. Used to present results to stakeholders.

**nexamart_analytics.sql**  
SQL queries for data analysis. Extracts insights from customer database.

**screenshot.jpg**  
Dashboard preview. Shows what the analysis looks like.
